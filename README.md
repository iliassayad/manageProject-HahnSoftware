# Project Task Manager - Task Management Application

A modern full-stack application to manage projects and their associated tasks, featuring JWT-based authentication.

## Features
- Account creation
- User authentication and authorization with JWT
- Create, edit, and delete projects
- Create, edit, and delete tasks within projects
- Track task status: To Do, In Progress, Completed, Cancelled
- Project progress tracking with completion percentage
- Responsive and user-friendly interface
- RESTful API for backend services

## Tech Stack
**Backend**
- Java 21 with Spring Boot 3.5.8
- Spring Security with JWT for l'authentication
- Spring Data JPA for data persistence
- PostgreSQL
- for dependency management and building the backend

**Frontend**
- React 19 with Vite for fast and modern frontend development  
- React Router DOM* for client-side routing and navigation  
- Axios for making API requests  
- Tailwind CSS for styling and responsive design  
- Lucide React for beautiful and consistent icons

## Installation & Run

- Clone the repository
- build images and run docker containers (docker-compose up --build)
